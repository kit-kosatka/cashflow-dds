# CashFlow DDS

Веб-приложение для учета доходов и расходов.

## Стек

- Python
- Django
- SQLite
- django-smart-selects

## Возможности

- Создание, редактирование и удаление операций.
- Управление статусами, типами операций, категориями и подкатегориями через Django Admin.
- Просмотр операций в виде таблицы.
- Фильтрация операций по:
  - дате;
  - статусу;
  - типу операции;
  - категории;
  - подкатегории.
- Каскадный выбор связанных данных:
  - категории зависят от выбранного типа операции;
  - подкатегории зависят от выбранной категории.
- Валидация бизнес-правил:
  - нельзя выбрать категорию, которая не относится к выбранному типу операции;
  - нельзя выбрать подкатегорию, которая не относится к выбранной категории.

## Установка и запуск

Клонировать репозиторий:

```bash
git clone https://github.com/kit-kosatka/cashflow-dds.git
cd cashflow-dds
```

Создать виртуальное окружение:

```bash
python -m venv .venv
```

Активировать виртуальное окружение.

Linux / macOS:

```bash
source .venv/bin/activate
```

Windows:

```bash
.venv\Scripts\activate
```

Установить зависимости:

```bash
pip install -r requirements.txt
```

Применить миграции:

```bash
python manage.py migrate
```

Создать администратора:

```bash
python manage.py createsuperuser
```

Запустить сервер:

```bash
python manage.py runserver
```

Админ-панель доступна по адресу:

```
http://127.0.0.1:8000/admin/
```

## Тестирование
